# ShorePilot 上岸作战舱

一个面向长期 DDL 和生活目标管理的安卓离线 App：目标、日历、学习/复习、体重运动、开支预算、日常事务和复盘统计放在同一个手机仪表盘里。

## 打开方式

1. 用 Android Studio 打开本文件夹。
2. 等待项目同步完成。
3. 连接安卓手机并开启 USB 调试，点击 Run 即可安装。
4. 或者在 Android Studio 里选择 Build > Build Bundle(s) / APK(s) > Build APK(s)，生成 APK 后发送到手机安装。

## v3 功能

- 首页：最近 DDL、目标概览、学习完成度、减重进度、预算余量。
- 目标：支持添加多个目标/DDL，例如考研、论文、比赛、实习申请；支持设置每周投入时长。
- 日历：按计划日期查看未来 7 天安排，支持提前规划下周待办。
- 复习：添加学习任务、绑定所属目标、记录预计时长、勾选完成。
- 本周目标：支持在复习页临时调整本周学习目标，例如这周只学 20h，下周再恢复。
- 运动：记录体重和运动，支持自定义减重目标，计算 7 日平均体重。
- 开支：记录当前余额、月收入、固定开支、存钱目标、意外开支，计算本月剩余预算和今日建议可花。
- 事务：记录日常任务、设置计划日期并打勾。
- 复盘：生成本周学习、运动、开支建议，显示最近 7 天趋势，并可导出全部数据。

数据全部保存在手机本地，不需要联网。
